Admin Guide

Use the scripts under scripts/ to build indexes, generate workspace data from files/, validate manifests, create a baseline snapshot, package a release, and run a local preview server.

  • python tests/run_all.py runs the current validation set.
  • Add new shared documents under files/single_page/, files/multi_page/, or files/resumes/ before regenerating the workspace data.
  • python scripts/opportunities/validate_opportunity_outputs.py validates the Venuiti opportunity layer and writes a summary under data/opportunities/venuiti/technology_project_manager/.
  • python scripts/opportunities/package_opportunity.py creates the deterministic opportunity bundle under .ai_downloads/.
  • python scripts/create_baseline.py creates a traceable snapshot in backups/.
  • python scripts/package_release.py creates a release zip in reports/packages/.
  • powershell -File scripts/run_local.ps1 starts a simple local server.
