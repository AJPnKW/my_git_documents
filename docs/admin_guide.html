Admin Guide

Use the scripts under scripts/ to build indexes, validate manifests, create a baseline snapshot, package a release, and run a local preview server.

  • python tests/run_all.py runs the current validation set.
  • python scripts/create_baseline.py creates a traceable snapshot in backups/.
  • python scripts/package_release.py creates a release zip in reports/packages/.
  • powershell -File scripts/run_local.ps1 starts a simple local server.
