Content Model

A central site registry defines the top-level areas. Each sub-site has a manifest and its own content folder. Resume postings use normalized artifact names and a posting metadata file for association.

  • Yellow House content lives under sites/yellow_house/.
  • Resume postings live under sites/resumes/companies/<company>/postings/<posting>/.
  • Required normalized posting files: job_posting.html, resume.html, notes.html, interview_questions.html, posting.json.
