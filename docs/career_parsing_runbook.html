Career Parsing Runbook

The current run ingests 251 source artifacts from the configured historical job-search roots and writes whole-file outputs under data/career/.

  1. Discover supported career-relevant files recursively from the configured source roots.
  2. Extract text deterministically from DOCX/XLSX/HTML/TXT/CSV/JSON and best-effort PDF content.
  3. Use the online-forms workbook as the canonical role and education spine.
  4. Enrich facts from supporting documents while preserving every source reference.
  5. Activate downstream Venuiti opportunity files through alias resolution to canonical master-story IDs.
  6. Validate JSON structure, story references, downstream activation state, and Python compilation for all scripts/career/*.py files.
  7. Generate the career master, story master, source index, entity timeline, activation summary, validation summary, logs, and zip bundle.

Operational command: python scripts/career/run_career_pipeline.py.
