User Workflows

The baseline supports two main flows: private document browsing and interview preparation with side-by-side document reading.

Yellow House

  • Open the site from the home page or the local summary page.
  • Pass the neutral gate if local auth config is present.
  • Open the investor analysis or supporting imported assets.

Resumes

  • Open Resumes, then Venuiti, then the posting workspace.
  • Open Interview Mode for 1/2/3 panel reading.
  • Capture notes in the bottom panel and copy them out locally.
